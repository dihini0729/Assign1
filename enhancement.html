Enhancements

  • Using parallax effect on the 4 subtopic pages.
    • Parallax is used to create an illusion of depth and motion by having multiple layers of images or content that move at different speeds as the user scrolls down the page.
  • Creating an extra web page named "About Us."
    • "About Us" web page gives information about the members of the group.
  • Using animation on the home page.
    • The cord for the web page: .card{ display: inline-block; border-radius: 10px; padding:15px 30px; cursor: pointer; margin: 10px 15px; transition:transform 0.5s; } .card img{ top:85px; width: 280px; height: 188px; border-radius: 20px; position:sticky; } .card:hover{ transform: translateY(50px); opacity: 0.8; }
    • This code can be used to create an aesthetically pleasing card layout with images that animate on hover, providing a more engaging user experience.
